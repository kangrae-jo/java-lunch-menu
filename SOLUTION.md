# 모델링

## 용어 정리

### 주문서

| 용어   | 설명                               |
|------|----------------------------------|
| Bill | 주문에대한 종합적인 정보가 들어있는 영수증 (출력에 사용) |

### 주문 품목

| 용어        | 설명                           |
|-----------|------------------------------|
| OrderItem | 주문을 구성하는 품목으로, 메뉴와 개수 정보를 가짐 |

### 주문

| 용어    | 설명                                     |
|-------|----------------------------------------|
| Order | 주문 품목이 모여서 한 사람이 주문한 모든 메뉴와 개수 정보를 나타냄 |

### 메뉴

| 용어   | 설명        |
|------|-----------|
| Menu | 주문 가능한 메뉴 |

### 카테고리

| 용어       | 설명                |
|----------|-------------------|
| Category | 각 메뉴의 종류가 되는 카테고리 |

### 할인

| 용어              | 설명            |
|-----------------|---------------|
| Discount        | 각 주문에 들어가는 할인 |
| Discount Period | 할인 종류별 적용 기간  |
| Discount Type   | 할인 종류         | 

### 혜택

| 용어             | 설명    |
|----------------|-------|
| Benefit Amount | 혜택 금액 |
| Benefit Menu   | 혜택 메뉴 |

### 배지

| 용어         | 설명                                             |
|------------|------------------------------------------------|
| Badge      | 주문 금액에 따라 차등 지급되는 배지                           |
| Badge Type | 배지 종류는 (5천 원 이상) 별, (1만 원 이상) 트리, (2만 원 이상) 산타 |

    - [ ] 이벤트 배지가 부여되지 않는 경우, "없음"으로 보여 주세요.
